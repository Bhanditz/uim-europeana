Europeana: Europeana Data Model (EDM) - disclaimer

The latest EDM specifications v5.2.2 and the EDM Primer are available at http://version1.europeana.eu/web/europeana-project/technicaldocuments/.

Any provider who wishes to resubmit data to Europeana in EDM or map their data to EDM, in order to increase its richness within Europeana, can refer to the EDM Mapping Guidelines. It contains definitions of the properties, information about the data types that can be used as values and the obligation level of each property. The mapping rules are defined against the EDM XML schema.

For further information, please contact info@europeana.eu (using “EDM” as subject) or look for more technical details on the EDM Prototyping Wiki at http://www.europeanalabs.eu/wiki/EDMPrototyping.
